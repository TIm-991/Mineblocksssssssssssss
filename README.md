# Mineblock Command Lobby - Neues XP- & Quest-System

Dieses README erklärt das neue Fortschrittssystem für alle Spieler.

## Überblick

Seit dem Rework gibt es **zwei getrennte XP-Systeme**:

- **Level-XP**: Für dein Account-Level (max. Level 200)
- **Battle-Pass-XP**: Für deine Pass-Stufe (max. Stufe 200)

Beide Systeme laufen parallel, sind aber unabhängig voneinander.

## Was ist neu?

- Battle Pass hat ein eigenes XP-System
- Quests wurden überarbeitet (Design + Rewards)
- Daily/Weekly Quests geben jetzt gemischte Belohnungen:
  - Level-XP
  - Pass-XP
  - M-Bucks
  - Skillpunkte (je nach Quest)
- Neue Progress-Anzeigen direkt im Quest-Menü

## Wo sehe ich meinen Fortschritt?

### Lobby > Progress & Modus

Dort siehst du:

- `Level`
- `Pass Stufe`
- `Pass-XP`

### Quest-Menü

Oben im Quest-Bereich siehst du:

- **Level-XP Track**
- **Battle-Pass-XP Track**

Damit erkennst du sofort, wie weit du in beiden Systemen bist.

## So funktionieren Quests jetzt

1. Quest auswählen (Daily oder Weekly)
2. Fortschritt bis Ziel erfüllen
3. Button `Rewards abholen` drücken
4. Belohnungen werden sofort verteilt

## Quest-Reward-Logik

Quests nutzen jetzt Reward-Pakete statt alter Einzel-Typen.

Mögliche Reward-Felder:

- `levelXp`
- `passXp`
- `mBucks`
- `skillPoints`

Weekly-Quests bekommen intern einen Extra-Boost auf XP/M-Bucks (stärkerer Progress).

## Beispiel

Eine Weekly-Quest kann z. B. geben:

- +420 Level-XP
- +320 Pass-XP
- +120 M-Bucks

Das heißt:

- Account-Level steigt durch Level-XP
- Pass-Stufe steigt durch Pass-XP
- Währung wird direkt gutgeschrieben

## Kurztext für Announcement

"Account-Level und Battle Pass sind jetzt komplett getrennt. Quests geben eigene Level-XP und Pass-XP. Ihr seht beide Fortschritte live in Lobby und Quest-Menü."

## FAQ

### Warum steigt mein Level, aber die Pass-Stufe nicht gleich schnell?

Weil beide Tracks getrennt skaliert sind. Pass-XP hat eigene Quellen und eigene Balance.

### Kann ich nur auf Battle Pass spielen?

Ja. Durch Quests und weitere Aktivitäten kannst du gezielt Pass-XP farmen.

### Gibt es ein Max-Level?

Ja:

- Account-Level: 200
- Battle Pass: 200
